# FFT2-SSIM-IC

## Setup

#### Conda 
```bash
  git clone https://github.com/sudharshan2001/FFT2-SSIM-IC.git
  cd FFT2-SSIM-IC
```

##### Create Environment using conda

```bash
  conda create -n fft_ic python=3.9.7
  activate fft_ic
```

##### Install the requirements
```bash
  pip install -r requirements.txt
```
